## OMI ADMIN

> 管理系统前端解决方案

[→ 点我体验](https://tencent.github.io/omi/packages/admin/dist/index.html)

**功能特性:**

* 基于 omi web components
* vite + ts + twind + lazy load
* 主题实时切换
* 路由动画
* 海量 ICON 
* 内置文档系统
* 可关闭打开的导航面板
* 多 tab 页面系统
